# Avoider — Simple 2D Game

**Avoider** is a fast-paced 2D browser game where you control a yellow block at the bottom of the screen and dodge falling obstacles. The game features multiple obstacle shapes, dynamic difficulty, and a sleek UI for an engaging experience.

## Features

* **Multiple Shapes:** Rectangles, circles, and triangles fall with varying speeds and sizes.
* **Dynamic Difficulty:** Adjust the challenge with a real-time difficulty slider.
* **Score & Dodges Tracking:** Each avoided obstacle increases your score and dodges count.
* **Intuitive Controls:** Move with arrow keys or A/D, jump with W, ↑, or Space.
* **Pause Functionality:** Pause/resume the game with the button or P key.
* **Lightweight & Browser-Based:** No installation needed; runs smoothly in any modern browser.

## How to Play

1. Open the `Simple-2d-avoid-game.html` file in a browser.
2. Use **← → or A/D** to move left and right.
3. Press **↑, W, or Space** to jump.
4. Adjust the **difficulty slider** to control the speed and spawn rate of obstacles.
5. Try to **survive as long as possible** while dodging obstacles.
6. Use the **Pause** button or press **P** to pause the game.
7. Check your **Score** and **Dodges** in the HUD below the instructions.

## Installation

No installation required. Simply clone or download this repository and open the HTML file in any modern browser.

```bash
git clone https://github.com/yourusername/Simple-2d-avoid-game.git
cd Simple-2d-avoid-game
open Simple-2d-avoid-game.html
```
